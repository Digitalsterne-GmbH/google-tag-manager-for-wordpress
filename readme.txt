== Description ==

Simple, secure and fast way to integrate Google Tag Manager into your Wordpress Installation.

== Installation ==

1. Install this Plugin via the integrated Installer or upload it in Wordpress
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Settings' > 'General' > 'Google Tag Manager' and set the ID of your Tag Manager Container.
